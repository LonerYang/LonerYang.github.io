Linux课堂之网络命令详解

  • Linux
  • 2019-03-29

一、网络相关命令

1、用ping命令测试Linux是否联网

ping[选项] [网址/ip] ping 网址还是会被解析成 ip 地址

选项：-c n Linux中 ping 默认是无限 ping ，可以使用 -c选择设置 ping 的次数，如果不使用 -c选项，无限 ping 可以使用 Ctrl + c 退出

eg：使用 ping -c 3 www.baidu.com 命令 ping 百度3次

2、使用 ip addr 查看网卡信息

eno16777736：网卡代号

inet 192.168.35.157： ip地址
inet6：IPv6的地址，还不怎么普及

lo：loop设备的意思，地址是127.0.0.1即本机回送地址

3、traceroute [网址] 显示本机到目标网址的路径

traceroute [网址]

4、查看网络状态

netstat [选项]

-t TCP 协议

-u UDP 协议

-l 监听

-r 路由

-n 显示 ip地址和端口号

常用的组合

4.1、netstat -tlun查看端口信息

4.2、netstat -an 查看本地所有网络连接

4.3、netstat -rn 查看本地路由网关

二、扩展命令

5、finger [用户名] 查看用户信息

eg：查看 root 用户信息

finger 相当于 env

env：查看当前用户的环境信息

6、echo输出命令

1、eg：打印字符串输出到控制台中 echo "aaa"

2、echo [信息] >> [文件名] ：追加内容到文件后面

​ echo [信息] > [文件名]：将内容覆盖文件中原有的内容

eg：通过echo bb >> a.txt 将字符串 bb 追加到 a.txt 文件中

eg：通过 echo cc > a.txt 将字符串 cc 覆盖 a.txt原有的内容

7、有关时间的命令

date ：查看当前系统时间

eg：

man date：查看帮助文档发现可以通过 date 加上格式就可以修改系统默认时间

MMDDhhmmCCYY.ss ===> 月/日/时/分/公元/年/.秒

eg：修改系统默认时间为2020年1月1号12时24分48秒

cal：日历

eg：cal -y 2019 查看2019年的整年的日历

eg：cal 3 2019 查看2019年3月的整月的日历

8、设置别名命令

alias：查看系统所有被设置别名的命令

eg：alias al = ’ls -i‘ 设置ls -i 命令的别名为al （重启后失效）

9、帮助文档命令

man [命令名称]

help [命令名称] 只能查看Linux内部命令（不是外部额外安装的），

使用 type [命令] 查看是否为内部命令

whereis [命令]：查看命令文件信息（查看到命令安装路径、查看到命令帮助文档的安装路径）

which[命令]：只能查看到命令的安装路径
